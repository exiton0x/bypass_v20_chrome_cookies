# Installation
Make sure you have **Rust** installed. If not, you can install it via this website [rust](https://www.rust-lang.org/fr/tools/install):       
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
Then, clone this repository and build the project:
```
git clone [https://github.com/yourusername/repository-name.git](https://github.com/exiton0x/decrypt_v20_chrome_cookies)
cd repository-name
cargo build --release
```
